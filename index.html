Acerca de mí

Soy ingeniero de ejecución en informática de profesión, 15 años de experiencia en el rubro de las telecomunicaciones, específicamente en el área de Monitoreo y Control, pero siempre me llamo la atención el desarrollo web, mi tesis fue un sistema web para almacenar documentos basado en PHP y MySql, pero mi andar con la programación tuvo una pausa larga en estos años, ya que mi trabajo se orienta principalmente en Networking. Pero ahora último me ha nacido nuevamente la necesidad de aprender programación, por lo cual en estos momentos me encuentro realizando un bootcamp de desarrollo web impartido por Practicum latam. Mi objetivo es poder desarrollar sistemas de forma autónoma, que presenten un servicio de real utilidad para quiénes lo utilicen.

Sprint 0

Para este proyecto del sprint 1, se pusieron en práctica todos los conocimientos adquiridos hasta el momento en el Bootcamp de Practicum de Desarrollo Web, en el módulo HTML y CSS avanzados - sprint 1, desde cómo abordar un proyecto, y como dividirlo en fragmentos más manejables, utilizar posicionamiento con flexbox, usar los comandos básicos de GitHub para control de versiones, y poner en práctica todo lo aprendido en HTML (etiquetas semánticas) y CSS (creando clases utilizando metodología BEM – Bloque, Elemento y Modificador).
